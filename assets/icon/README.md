# App Icon Assets

Letakkan file icon aplikasi di folder ini:

1. **app_icon.png** - Icon utama aplikasi (1024x1024 px, PNG dengan transparency)
2. **app_icon_foreground.png** - Layer foreground untuk adaptive icon Android (1024x1024 px)

## Instruksi Lengkap
Lihat panduan lengkap di `docs/APP_ICON_GUIDE.md`

## Catatan
Icon saat ini menggunakan default Flutter icon. Ganti dengan icon wallet/dompet yang sesuai dengan splash screen.
